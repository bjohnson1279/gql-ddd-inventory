## 2025-02-18 - Single-Purpose Tool Input AutoFocus
**Learning:** Single-purpose tool tabs (like Barcode Scanners or Serial Tracers) benefit greatly from `autoFocus` on their primary input field. Without it, users must redundantly click to activate the input after navigating to the tab.
**Action:** Always add `autoFocus` to the primary input field of dedicated tool views to inherently grab focus on mount.
